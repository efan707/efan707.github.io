CSS选择器优先级判定

日期: 2014-11-22   |   分类: CSS   |  

CSS不同方式引入

CSS优先级即CSS样式在浏览器中被解析出来的先后顺序。
对于不同方式引入的样式，一般情况下，优先级如下：
外部样式(External style sheet) < 内部样式(Internal style sheet) < 内联样式(Inline style)
——如果外部样式放在内部样式的后面，则外部样式将覆盖内部样式；JS设置的样式为内联样式。
总之一句话：

样式位置愈近之，优先级愈高之

选择器权重

约定俗成，我们用(A,B,C,D)来表示CSS选择器的权重，其中：
A表示CSS是否为内联样式，是为1，否为0
B表示ID选择器的个数
C表示类、伪类和属性选择器的个数
D表示标签、伪元素的个数（例如，p:first-letter 这样的选择器，计算为0, 0, 0, 2）。
具体如下表所示：

选择器类型 权重(A,B,C,D) 例子
内联样式 (1,0,0,0)
ID选择器 (0,1,0,0) #IDName{}
CLASS类选择器 (0,0,1,0) .className{}
伪类选择器 (0,0,1,0) :hover{}
属性选择器 (0,0,1,0) [href]{}
标签选择器 (0,0,0,1) p{}

优先级比较规则：从权重左边的第一位开始比较大小，如果较大，则优先级较高；反之，则优先级较低；如果一样，继续往右逐位比较。(IE不支持!important)

权重优先级：important > 内联 > ID > 类 | 伪类 | 属性选择 > 标签 | 伪对象 > 继承 > 通配符

DOCTYPE声明

DOCTYPE标签是一种标准通用标记语言的文档类型声明，它的目的是要告诉标准通用标记语言解析器，它应该使用什么样的文档类型定义（DTD）来解析文档。
<!DOCTYPE> 声明必须是HTML文档的第一行，位于标签之前。在HTML4.01中，<!DOCTYPE>声明引用DTD，因为HTML4.01基于 SGML。DTD规定了标记语言的规则，这样浏览器才能正确地呈现内容。
