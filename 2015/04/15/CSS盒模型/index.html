css盒模型及元素分类

日期: 2015-04-15   |   分类: CSS   |  

CSS中， Box Model叫盒子模型（或框模型），Box Model规定了元素框处理元素内容（element content）、内边距（padding）、边框（border） 和 外边距（margin） 的方式。盒模式有两种：W3C规定的标准盒模型和IE盒模型。
IE盒模型是较早版本（IE5.5及之前）IE，以及怪异模式下的IE6及更高版本的渲染模式。

W3C规定的盒模型下，width和height只是内容部分的宽高，不包括padding、border；而IE盒模型下，width、height包括了padding和border。

IE盒模型

应用Wiki中对于IE盒模型的描述：

IE盒模型是指早期版本的Internet Explorer调整网页元素大小的方法，和W3C推荐的标准方式不同。在Internet Explorer 6中，浏览器支持一种解决了这种差异的可选的渲染模式（叫做“标准模式”）。出于向后兼容的原因，所有版本的IE（截至IE 9及IE 10 Developer Preview）仍然默认表现为非标准的模式。

为了使盒模型遵循W3C标准，在IE6以及之后的版本中，在标准兼容模式（standards compliant mode）下使用的是W3C的盒模型标准。在IE5.5及更早版本中，则需要采用其他的技术手段,在此不赘述。
此外，CSS3中还提供了box-sizing 属性,用于对盒模型的设定，在此就不做展开了。
