CSS Reset 与 CSS Normalize

日期: 2015-07-01   |   分类: CSS   |  

浏览器差异万千，为了尽可能地统一，我们常常引入CSS Reset或Normalize.css，至于其中的内容及区别，下面述之。

CSS Reset

CSS Reset的思想就是先破后立，不管你是IE，还是Chrome，先统统reset到同样的方式，话说好期待浏览器的大一统时代。CSS Reset样式一搜一箩筐，有淘宝的、雅虎的、各位大神的，关于如何选取，还是一句话精辟-不选最贵的，只选适合自己的。实际项目中，由于添加过多的reset样式会影响渲染性能，所以要写一份适合自己项目的reset。而由于各版本reset样式大差不差，所以先选取一份，然后通过增、删、改，量身定做一份属于自己的reset。

Normalize.css

如果说CSS reset是激进派，那么Normalize.css就是柔和派。CSS reset通过粗暴的方式来达到统一，属于革命；而Normalize.css保留浏览器的原来样式并且做到每个浏览显示一致，是改革。Normalize.css 只是一个很小的CSS文件，但它在默认的HTML元素样式上提供了跨浏览器的高度一致性。相比于传统的CSS reset，Normalize.css是一种现代的、为HTML5准备的优质替代方案。关于它的用法，网络上有很多详细的文档，也可从github上下载其源码一睹其风采。
