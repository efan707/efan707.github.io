何为Git

Git是目前世界上最先进的分布式版本控制系统（没有之一）。它不需要服务器端软件，就可以運作版本控制，使得源代码的发布和交流极其方便。Git的速度很快，最为出色的是它的合并追踪能力。同时，在GitHub上，人们可以利用Git极其强大的克隆和分支功能，与世界人民一道，自由欢快地参与各种开源项目。点击Git下载Git for windows。

Git your github

这个世界上有个叫GitHub的神奇的网站，这个网站就是提供Git仓库托管服务的。建立本地git与远程Github连接需要SSH协议，连接的过程参照Github&Hexo搭建blog。操作Github的主要命令有：

$ git remote add origin git@github.com:efan707/learngit.git   //本地与远程仓库相关联
$ git push -u origin master    //第一次把本地的master分支内容推送的远程的master分支并关联
$ git push origin master  //把本地master分支的最新修改推送至GitHub
$ git clone git@github.com:efan707/learngit.git    //克隆远程库
$ git clone git@github.com:efan707/learngit.git -b branchName    //克隆远程分支
