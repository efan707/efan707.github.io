CSS画图形

日期: 2014-12-06   |   分类: CSS   |  

利用CSS，可以画出各式各样的图形，下面就列举一二示之。

不同边界样式

通过对border属性的设置，可以产生一些另类的效果。

梯形


三角形

通过设置边框显示与否，可以形成各个方向的三角形。

空心三角形

通过设置两个三角形，after形成的三角形定位到原三角形内部，并设置颜色为背景色，形成“空心”。

五角星

CSS3还能实现更为复杂的图形。
五角星的画法其实也比较简单，主要是由三个等腰三角形拼凑而成(两大一小)，当然也可以借由其他方式进行整合。

心形


八卦

八卦的思路如下所示：

#ba-gua{
       width: 96px;     height: 48px;     
       background: #eee;     
       border-color: red;     
       border-style: solid;    
       border-width: 2px 2px 50px 2px;     
       border-radius: 100%;     
       position: relative;
}
 #ba-gua:before {     
    content: "";     
    position: absolute;     
    top: 50%;     left: 0;     
    background: #eee;    
    border: 18px solid red;     
    border-radius: 100%;     
    width: 12px;     height: 12px; }   
#ba-gua:after {    
     content: "";     
     position: absolute;     
     top: 50%;     left: 50%;     
     background: red;     
     border: 18px solid #eee;     
     border-radius:100%;     
     width: 12px;     height: 12px; }
