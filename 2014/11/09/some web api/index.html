Some web API

日期: 2014-11-09   |   分类: 前端杂烩   |  

前言

搭建web的过程中，可能需要一些常见的应用，譬如地图、天气、搜索等等。而在这时，一些第三方API便让我们能方便地实现所需的功能。国内的API可参照BaiduAPIStore或DevStore。下面就结合实例，列举二三。

地图

常用的有以下一些，考虑Google容易被墙，推荐使用百度地图。百度地图的API比较齐全，文档介绍也比较详尽，易上手。最喜欢百度的卫星地图，比较清晰，没有其他地图的灰蒙之感。
谷歌地图、必应地图、OpenStreetMap、百度地图、图吧、我要地图、搜狗地图、高德地图

天气

可以通过一些天气网站的API获得关于天气的JSON或XML数据，然后便可以通过JS来进行处理，关于网页中数据的处理会有篇幅进行专门说明。如果需要快捷开发的话，也可以借助类似心知天气插件这样直接现成的货。

下面是一些天气接口：
1、金山用的JSON数据
http://weather.123.duba.net/static/weather_info/101121301.html
2、小米用的JSON数据
http://weatherapi.market.xiaomi.com/wtr-v2/weather?cityId=101121301
3、中央天气预报
http://weather.51wnl.com/weatherinfo/GetMoreWeather?cityCode=101040100&weatherType=0
http://weather.51wnl.com/weatherinfo/GetMoreWeather?cityCode=101040100&weatherType=1
4、未知来源的数据接口
http://wthrcdn.etouch.cn/weather_mini?city=北京
通过城市名字获得天气数据，json数据
http://wthrcdn.etouch.cn/weather_mini?citykey=101010100
通过城市id获得天气数据，json数据
http://wthrcdn.etouch.cn/WeatherApi?citykey=101010100
通过城市id获得天气数据，xml文件数据,当错误时会有error节点
http://wthrcdn.etouch.cn/WeatherApi?city=北京
通过城市名字获得天气数据，xml文件数据
